• Component Selection Tool

Introduction

The following is a tool developed to aid engineers in appropriate component selection. To do so, a list of pump, valve, and actuator types has been compiled and their key attributes and functionality noted. In the case of valves, the below inputs allow the user to specify criteria regarding the valves desired functionality, generating a refined list of suitable valves for the particular appication. Additional information, including common actuators, can be found within each of the linked pages. Similar pages have been included with information on methods of connection and pump types.

This tool has been designed as a guide only. The intention is not to output the exact valve appropriate for the specific application. Instead, this is intended as a first step in component selection, directing the user to possible options. From here, the exact component and material selection will require further engineering judgement in order to complete the selection process.
